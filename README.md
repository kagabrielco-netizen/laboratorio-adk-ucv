# Laboratorio: Agente IA con Integración Continua - UCV

Proyecto de automatización y desarrollo de un Agente de Inteligencia Artificial utilizando la API de Google GenAI, evaluado con calidad de código mediante SonarQube Cloud y desplegado con GitHub Actions.

## 🚀 Nuevas Características (Reto Alumno)
1. **5 Conceptos Técnicos Añadidos:** CI/CD, SonarQube, GitHub Actions, Pruebas Unitarias y Poetry de manera integrada.
2. **Nueva Tool (`calcular_promedio`):** Calcula promedios con validaciones estrictas de notas entre 0 y 20.
3. **Pruebas Unitarias:** Implementadas con `pytest` y configuradas mediante `conftest.py` para garantizar el correcto funcionamiento del core.
4. **Validación de Entradas:** Filtros de seguridad que evitan errores si las notas están vacías o fuera de rango.

## 🛠️ Tecnologías Utilizadas
* Python 3.12
* Poetry (Gestión de dependencias)
* Google GenAI SDK
* GitHub Actions (Pipeline CI)
* SonarQube Cloud (Calidad de código)
* Pytest (Pruebas de software)
